Development of fully functional Arduino GPS Device (NEO-6M) in 2018

Decoding Satellite data format (NMEA) to extract meaningful information of specific location using GIS Hardware as a Semester project


The Arduino GPS project set out to create a tracking system by utilizing the NEO-6M GPS module alongside the Arduino Uno. This report chronicles the steps involved in linking the GPS module, acquiring raw GPS NMEA data, parsing the data to extract essential location details, and provides the project's historical context. The primary focus was on extracting key information from NMEA, including "LATITUDE, LONGITUDE, PDOP, ELEVATION, and NUMBER OF SATELLITES."

John Hopkins University COVID Time Series Web App 2020

Development of app using Arcgis Online for Patient status visaulization in reference to time


Johns Hopkins University leveraged the ArcGIS Online Time Series web app to visualize COVID-19 patient data, enriched by datasets curated through my freelance GIS work on Upwork. Collating information from Excel and various documents, I crafted a foundation for this dynamic tool. Our collaborative effort transformed raw data into an interactive platform, empowering researchers to monitor trends and identify infection rate fluctuations. This initiative highlighted the synergy between data organization, technological innovation, and visualization techniques in addressing the global health crisis.
